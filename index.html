Portfolio

Innate Solutions, Inc (Innate) has designed and is developing an integration solution software product called i Logistic Integrator. Its our proprietary product.

i Logistic Integrator is an application Integration project enable the information exchange between GLS (Global Logistics solution) and Oracle EBS and eliminate the need for keying the information into both the systems, which then eliminates the data entry errors. The core integration points that are included in this product are Compliance Requests, Item Restriction Check, Shipment creation, Ship processing, Void Shipment, Update Shipment, Generate PDF Shipment Document Request, EOD carrier processing, Service Payments Accessorial, Rate Shop Service, Create freight lines in EBS with Tracking Number from GLS.
